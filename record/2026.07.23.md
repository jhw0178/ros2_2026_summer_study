#### 3일차 복습
##### (1) DDS 설정
    - wsl과 통신하기 위해 mirror-net을 활용하고 방화벽을 허물어야 함
    - UDP를 기본적으로 사용하며, TCP를 사용하려면 DDS 설정을 건드려야 함
##### (2) 인터페이스
    - ros2에서 기본적으로 제공하는 인터페이스 존재 but, 유저에게 맞춰 쓰는 데 한계
    - rosidl(ROS interface definition language)를 활용해서 유저에게 맞는 인터페이스 생성
##### 토픽과 서비스
    - Topic은 Publihser와 Subscriber를 통해 통신 -> 비동기식 단방향
    - Service는 Client와 Server를 통해 통신하며, Request 영역과 Response 영역이 존재 -> 동기식 양방향
